# videoVigilanciaDesktopPython
instalar python
```
sudo apt-get install python3
```
instalar pygame
```
sudo apt-get install python3-pygame
```
si tiene python 2.X
```
sudo apt-get install python-pygame
```
otras dependencias

```
sudo apt-get update
sudo apt-get install python-pygame
sudo apt-get install python-picamera
```
ejecutar
```
python app2.py
```
